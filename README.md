# Visitor Map Dashboard

This repository is a standalone visitor-insights page that displays a live visitor map and supporting explanatory content.

## What it is

A polished dashboard that shows a live visitor map, explains what it tracks, and gives visitors a clear way to understand the site's reach.

## Why a separate repository

GitHub README files are great for a small image-based visitor badge, but they are not ideal for an interactive dashboard. A separate HTML/CSS/JS repository gives you more control over layout, spacing, and future features.

## Embed recommendation

- Use the image-based embed in a README.
- Use the JavaScript-based embed in this dashboard page.

## Suggested repository names

- `visitor-map-dashboard`
- `portfolio-visitor-map`
- `entomika-visitor-map`
- `website-visitor-insights`

## Files

- `README.md` — introduction layout.
- `index.html` — dashboard layout.
- `styles.css` — visual styling.
- `script.js` — interface behavior.

## Purpose

This page is meant to:

- Show a live geographic view of website visitors.
- Explain what the map represents.
- Give visitors and the owner a quick sense of audience reach.
- Provide a base for future analytics features.
